源代码库

### 代码目录说明

* [addons](./addons/README.md): 系统插件
* [api](./api/README.md): API服务
* [apps](./apps/README.md): 服务端程序 
* [settings](./settings/README.md): 配置管理器 
* [sql](./sql/README.md): 数据库脚本
* [styles](./styles/README.md): 样式库文件
* [utils](./utils): 实用程序

### 代码规范
